About CSET 3100

CSET 3100, or Advanced Website Design, is a course taught by Hong Wang that covers: HTML lists, HTML tables, HTML forms, Cascading Style Sheets (CSS), HTML5. The creation of static and animated web graphics. An introduction to dynamic website construction through the use of CGI/Python (server-side) and JavaScript (client-side) is provided and other contemporary topics such as website evaluation and web ethics.
